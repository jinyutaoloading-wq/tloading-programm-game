# 互动短剧游戏平台 Demo

这是一个单文件静态 MVP demo，用于验证互动短剧 H5 用户端与配置后台的基础闭环。

## 功能范围

- H5 用户端：首页、剧目详情、播放互动、剧情图、付费解锁模拟、结局反馈。
- 配置后台：项目工作台、配置总览、节点编辑器、审核发布、数据看板。
- Mock 状态：节点、选择、信任变量、存档、付费权益、发布状态。

## 部署

仓库已包含 GitHub Pages Actions 工作流。推送到 `main` 后会自动部署。
